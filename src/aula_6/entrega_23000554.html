Exercício 1 

public class InsertionSort {

    public static void insertionSort(int[] array) {
        for (int i = 1; i < array.length; i++) {
            //2 + 3*(n + 1) + 4n operações
            int key = array[i];
            // 5n operações
            int j = i - 1;
            // 4n operações
            while (j >= 0 && array[j] > key) {
                // 4*(n+1)^2 + 5*(n+1)^2 operações

                array[j + 1] = array[j];
                // 5n^2 + 4n^2 operações
                j = j - 1;
                // 4n^2 operações
            }
            array[j + 1] = key;
            // 6n operações
        }
    }

    public static void main(String[] args) {
        int[] array = {12, 11, 13, 5, 6};

        System.out.println("Array antes da ordenacao:");
        for (int num : array) {
            System.out.print(num + " ");
        }

        insertionSort(array);

        System.out.println("\n\nArray depois a ordenacao:");
        for (int num : array) {
            System.out.print(num + " ");
        }
    }
}

Exercício 2

Análise de Complexidade:

Esse algoritmo tem o seguinte número de operações:

  • Linha 1: for (int i = 1; i < array.length; i++)

    • 2 + 3*(n + 1) + 4n operações

  • Linha 2: int key = array[i];

    • 4n operações

  • Linha 3: int j = i - 1;

    • 4n operações

  • Linha 4: while (j >= 0 && array[j] > key) {

    • 4*(n+1)^2 + 5*(n+1)^2 operações

  • Linha 5: array[j + 1] = array[j];

    • 5n^2 + 4n^2 operações

  • Linha 6: j = j - 1;

    • 4n^2 operações

  • Linha 7: array[j + 1] = key;

    • 6n operações

Equação final:
T(n)=14+40∗n+22∗n2=>O(n2)T(n) = 14 + 40*n + 22*n^2 => O(n^2)T(n)=14+40∗n+22∗n2=>O(n2)
